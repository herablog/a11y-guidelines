### 2.1.2 キーボード操作を可能にする

#### 実装方法

##### 悪い実装例

例：モーダルウィンドウの場合

Tabキーを使えないようにし、クリックでのみクローズ出来る。

```html
<button type="button" onclick="onClick()" tabindex="-1">ボタン</button>
```

```javascript
function onClick() {
  modal.close();
}
```

##### 良い実装例

例：モーダルウィンドウの場合

Tabキーでボタンにフォーカスが移動出来る。
ESCキーでモーダルウィンドウをクローズ出来る。

```html
<button type="button" onclick="onClick()">ボタン</button>
```

```javascript
function onClick() {
  modal.close();
}

// キーボードイベントの実装
function onKeyup (e) {
  // ESCキー押下
  if (e.keyCode === 27) {
    modal.close();
  }
}
```
